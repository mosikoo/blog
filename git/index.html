4.1 git hook

githook实现自动化部署

在服务器端建立裸仓库

cd /opt/code
git init --bare test.git

修改hook,主要是修改post-receive用于接收成功后的操作

cd /opt/code/test.git/hooks
touch post-receive

编辑post-receive

vim post-receive

# post-receive内容
#!/bin/bash
DeployPath="/var/www/blog" #实际运行地址
cd $DeployPath
git fetch --all
git reset --hard origin/master

#退出vim修改权限
chmod +x post-receive

客户端操作

git clone

git clone git@IP:/opt/code/test.js

随意提交会发现钩子已经运行

本地也可以模拟

自己建立裸仓库

cd /Users/Swing
git init --bare test.git
# 然后与上面一样建立并修改钩子

建立客户端

cd /Users/Swing
git clone ./test.git #克隆本地git
cd test
# 随意操作git并提交内容会发现githook已运行
